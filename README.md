# # Shipment Performance Analyticsgit 

An end-to-end supply chain analytics project using Python, SQL, and Power BI.

## Dataset
[DataCo Smart Supply Chain](https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis)
— 180,000+ orders across global logistics operations.

## Project Phases
- **Phase 1 — ETL Pipeline** (Python): Load, clean, and store the data
- **Phase 2 — SQL Analysis**: KPI queries and metrics
- **Phase 3 — ML Models**: Demand forecasting and late-delivery prediction
- **Phase 4 — Power BI Dashboard**: Interactive reporting

## Setup

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/dataco-supply-chain.git
cd dataco-supply-chain

# 2. Create a virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Add the dataset
# Download DataCoSupplyChainDataset.csv from Kaggle
# Place it inside  data/raw/
```
